## Prerequisites:

- An AWS Account.
- AWS Access and Secret Key. Hint: Create an IAM role that allows programmatic access.
- Configure your IAM user on Visual Studio Code.
- Basic knowledge of AWS services and Terraform.
- Terraform version- v1.2.8
